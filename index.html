Startseite

Auf GitHub Pages liegt deine Seite meist unter /REPO/. Darum wird hier automatisch ein passendes <base> gesetzt, damit Assets und Partials überall funktionieren.

Teste es

  1. Öffne diese Seite im Browser.
  2. Klicke auf „Über uns“.
  3. Ändere dann in partials/header.html z. B. den Logo-Text oder einen Menüpunkt.
  4. Reload – Änderung ist auf allen Seiten sichtbar.
